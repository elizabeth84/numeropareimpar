###Ejercicio17.2- par e impar

##### Contexto

Escribir un algoritmo que indique al usuario si el número es par o impar.
Caso de Prueba 1
Input
num = 12
Output
"Par"
Caso de Prueba 2
Input
num = 33
Output
"Impar"
 
##### Pseudocodigo

1. Inicio
2. Ingrese num
3. ¿n es par?
->si -> Imprimir "n es par"
->no -> Imprimir "n es impar"
4. Fin
